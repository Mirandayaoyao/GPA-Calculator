author: Miranda Chen

A python-based lightweight calculator to convert GPA.    
Notice: Nanjing University adapt 5.00 GPA, while National Taiwan University with 4.30 GPA.  


## Mode
mode | GPA version
------------ | -------------
std4| standard 4.0
v1| version 1 of 4.0
v2| version 2 of 4.0
pku4| Peking University version of 4.0
